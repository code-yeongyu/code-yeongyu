    - '카카오톡 개발자는 저 말을 들을 때 마다 얼마나 뿌듯할까!' 라는 생각이 들었습니다.
    - *덕분에 **사람들과 가까운 서비스를 개발하는 꿈**을 갖고 있습니다.*
